입력된 `validated_pages`를 분석하여, **주식 투자를 갓 시작한 사회초년생**이 이해하기 어려울 법한 용어를 추출하세요.

## 페르소나 정의 (Target Audience)
1.  **나이/성향**: 20~30대 사회초년생. 투자를 이제 막 시작함.
2.  **지식 수준**:
    - ❌ **모름**: PER, PBR, EPS, 캔들차트, 이평선, 수급, 펀더멘털, 조정, 차익실현, 모멘텀 등.
    - ✅ **앎**: 매수, 매도, 주가, 돈, 이익, 월급 정도의 상식.

## 추출 기준
1.  **적극적 추출**: "이것도 모를까?" 싶으면 무조건 추출하세요. (예: PER, 양봉, 음봉, 시가총액 등 기초 용어 포함)
2.  **명사구(Noun Phrase) 우선**:
    - 단일 명사보다 **문맥이 포함된 명사구**를 우선 추출하세요.
    - 예: "수급" (X) -> "외국인 수급" (O)
    - 예: "조정" (X) -> "기간 조정" 또는 "가격 조정" (O)
    - 예: "저항" (X) -> "심리적 저항선" (O)
3.  **한자어/업계 은어**: "차익실현", "물량 소화", "오버행" 같은 표현도 추출하세요.
4.  **수량 제한**: 각 페이지(step)당 **최대 4개**의 핵심 용어를 추출하세요.
5.  **문맥 추출**: 해당 용어가 사용된 **원문 문장 전체**를 `context_sentence`에 담으세요.

## 출력 스키마 (JSON)
```json
{
  "terms_to_search": [
    {
      "step": 1,
      "term": "기간 조정",
      "context_sentence": "주가가 단기간 급등하여 기간 조정을 거치고 있습니다."
    },
    {
      "step": 1,
      "term": "PER",
      "context_sentence": "동종 업계 대비 PER이 낮아 저평가 매력이 부각됩니다."
    },
    {
      "step": 2,
      "term": "외국인 수급",
      "context_sentence": "최근 외국인 수급이 개선되며 주가 상승을 견인하고 있습니다."
    }
  ]
}
```
* `terms_to_search`는 모든 페이지에서 추출된 용어를 하나의 리스트로 담습니다.
* 중복된 용어는 제거하세요.
